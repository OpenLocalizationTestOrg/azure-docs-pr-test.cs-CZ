# <a name="get-started-with-the-sendgrid-connector"></a>Začínáme s konektor sendgrid vám umožňuje
Poskytovatel připojení SendGrid umožňuje odesílání e-mailů a správu seznamů příjemců.

Můžete začít s vytvářením aplikace logiky teď najdete v tématu [vytvoření aplikace logiky](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-sendgrid"></a>Umožňuje vytvořit připojení k sendgrid vám umožňuje
K vytvoření aplikace logiky sendgridu, musíte nejdřív vytvořit **připojení** pak zadejte podrobnosti pro následující vlastnosti: 

| Vlastnost | Požaduje se | Popis |
| --- | --- | --- |
| ApiKey |Ano |Zadejte svůj klíč k rozhraní API SendGrid. |

> [!INCLUDE [Steps to create a connection to SendGrid](../../includes/connectors-create-api-sendgrid.md)]
> 


Po vytvoření připojení, můžete ke spouštění akcí a naslouchat aktivačních událostí.

## <a name="connector-specific-details"></a>Podrobnosti o konkrétní konektor

Zobrazit všechny aktivační události a akce definované v swagger a také zobrazit žádné limity v [connector – podrobnosti](/connectors/sendgrid/).
