# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>Povolit prostředí preview ke správě vaší spravované domény z portálu Azure

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>Stávající spravované domény není uveden v prostředí portálu Azure (preview)
Máte stávající [Azure AD Domain Services](active-directory-ds-overview.md) spravované doméně, která byla vytvořena pomocí portálu Azure classic. Když přejdete na portál Azure, v okně Azure AD Domain Services (preview) není uveden tento existující spravované domény.

![Existující spravované domény není zobrazený na portálu Azure](./media/getting-started/missing-domain-services-instance.png)

Je to proto, že stávající spravované domény nebyl přesunuli do nového portálu Azure. Vyžádejte povolit nové rozhraní portálu Azure (preview) vaší existující spravované domény.


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>Připojení k verzi preview pro Azure prostředí portálu pro Azure AD Domain Services
Postupujte podle pokynů [kontaktujte nás](active-directory-ds-contact-us.md) stránky a e-mailu nám. Je nám ctí povolit Azure portálu preview vaší existující spravované domény.

> [!NOTE]
> Pokud nové prostředí Azure portálu pro Azure AD Domain Services je všeobecně dostupná (GA), všechny existující spravované domény automaticky migrovat. V tomto okamžiku budete moct spravovat vaší existující spravované domény z portálu Azure.
