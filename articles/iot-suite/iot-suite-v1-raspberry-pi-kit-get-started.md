# <a name="connect-your-microsoft-azure-iot-raspberry-pi-3-starter-kit-to-the-remote-monitoring-solution"></a>Připojit vaše Microsoft Azure IoT malin pí 3 Starter Kit k řešení vzdáleného monitorování

Kurzy v této části vám pomůže naučit se připojit k řešení vzdáleného monitorování zařízení malin pí 3. Vyberte vhodné preferovaný programovací jazyk kurzu a jestli máte k dispozici pro použití s vaší malin pí senzor hardwaru.

## <a name="the-tutorials"></a>Kurzů k

| Kurz | Poznámky | Jazyky |
| -------- | ----- | --------- |
| Simulovanou telemetrii (Basic)| Simuluje snímač data. Používá samostatný malin pí. | [C][lnk-c-simulator], [Node.js][lnk-node-simulator] |
| Skutečné senzor (střední) | Používá data ze senzoru BME280 připojené k vaší malin pí. | [C][lnk-c-basic], [Node.js][lnk-node-basic] |
| Aktualizace firmwaru implementace (Upřesnit)| Používá data ze senzoru BME280 připojené k vaší malin pí. Umožňuje při aktualizacích firmwaru vzdálené na vaše malin pí. | [C][lnk-c-advanced], [Node.js][lnk-node-advanced] |
