# <a name="using-the-deep-learning-virtual-machine"></a>Pomocí hloubkového učení virtuálního počítače

Po zřízení vaší hloubkové Learning virtuálního počítače (DLVM), můžete začít vytvářet modely hluboké neuronové sítě k vytváření aplikací AI v doménách jako počítač vize, znalosti jazyka. 

Existuje několik nástrojů uvedených na DLVM AI. [Stránka rozhraní hloubkové učení a AI](dsvm-deep-learning-ai-frameworks.md) obsahuje podrobnosti o tom, jak pomocí těchto nástrojů. [Ukázky stránky](dsvm-samples-and-walkthroughs.md) poskytuje odkazy na příklady kódu, který je předem zavedené ve virtuálním počítači pro každou z rozhraní vám pomohou rychle začít. 

# <a name="deep-learning-tutorials-and-walkthroughs"></a>Hluboké učení kurzy a návody

Kromě ukázky na základě framework poskytujeme také sadu komplexní návodů, které mají byl ověřen na DLVM. Tyto nápovědy vám rychle začít s prací váš vývojový hloubkové learning aplikací v doménách jako bitovou kopii a pochopení text/jazyk. Abychom mohli pokračovat v různých doménách a technologie přidat další kurzy začátku do konce.   

1. [Spuštění neuronové sítě pro různé architektury](https://github.com/ilkarman/DeepLearningFrameworks): Toto je komplexní návod, který ukazuje, jak migrovat kód z jednoho rozhraní. Také ukazuje, jak porovnat modelu a spustit doba mezi architektury. 

2. [Postupy: Průvodce k vytvoření řešení začátku do konce ke zjištění produkty v rámci bitové kopie](https://github.com/Azure/cortana-intelligence-product-detection-from-images): detekce bitové kopie je technika, můžete vyhledat a klasifikovat objekty v rámci bitové kopie. Tato technologie se může mají být předány obrovské výnosu mnoho reálného života obchodní domény. Například prodejců slouží tento postup k určení, který produkt a zákazník má převzata z police. Tyto informace pomáhají zase úložiště spravovat inventář produktů. 

3. [Klasifikaci dokumentů text](https://github.com/anargyri/lstm_han): Tento návod ukazuje, jak vytvářet a cvičení dva různé neuronové sítě architektury (hierarchické pozornost sítě a dlouho paměti krátké termín (LSTM) síť) pro klasifikaci dokumentů text pomocí rozhraní API Keras hloubkové informací. Keras je front-end na tři nejoblíbenější hloubkového učení architektury, Microsoft kognitivní Toolkit, TensorFlow a Theano.
