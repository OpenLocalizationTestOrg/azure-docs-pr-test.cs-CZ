# <a name="get-help-from-machine-learning-live-chat-support"></a>Získání nápovědy z podpory služby Machine Learning formou živého chatu
[Azure Machine Learning Studio](what-is-ml-studio.md) poskytuje intuitivní rozhraní pro vytváření modelů machine learning. Je [Galerie](gallery-how-to-use-contribute-publish.md) příklady přispěli komunity můžete začít pracovat. A dojde [fórum](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning) kde můžete klást otázky, abychom vás přejdete. 

Ale v některých případech stačí rychlé zeptejte můžete odblokovat. V [Machine Learning Studio](http://studio.azureml.net/Home) Hledat v horním navigačním panelu na ikonu konverzace.  Pokud se zobrazí tato ikona, která znamená, že člen týmu produktu je online, můžete v reálném čase.

![Azure ML chatu](./media/live-chat/AzureMLChatNavBar.png)

Zadejte dotaz a získat odpovědi!

![Dialogové okno konverzace Azure ML](./media/live-chat/AzureMLChat.png)

**Ikona Live Chat nezobrazuje?**

Live Chat pracovníci členové týmu pro Machine Learning. Pokud se nezobrazí ikona živé konverzace je to, protože není aktuálně k dispozici týmu. Například může být mimo běžnou pracovní dobu. 

Můžete vždy zveřejněte svoje otázky na [Machine Learning fórum](https://social.msdn.microsoft.com/forums/azure/home?forum=MachineLearning), kde bude vidět členové tým Machine Learning a dalších zkušení datových vědců a vývojářů.
