* Opravené havárií na privátní procházení (Safari).
* Opravené havárie v prohlížečích se soubory cookie zakázána.

* Vydání Azure Mobile Engagement.
* Aktualizovat tak, aby `appId` konfigurace je nahrazena konfiguraci řetězec připojení.

* Odebrat rozhraní API, která odesílá a přijímá zprávy mezi zařízeními.

* Odebrané podporu pro funkci Reach.
