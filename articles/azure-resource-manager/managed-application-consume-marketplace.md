# <a name="consume-azure-managed-applications-in-the-marketplace"></a>Využívat Azure spravované aplikace na webu Marketplace

Jak je popsáno v [spravovaných aplikací – přehled](managed-application-overview.md) článek, přihlašování a provést tak kompletní existují dva scénáře. Jeden je vydavatele nebo dodavatele, který chce vytvořit spravované aplikace pro použití zákazníků. Druhá je koncového zákazníka nebo příjemce spravované aplikace. Tento článek se zabývá druhý scénář. Popisuje, jak můžete nasadit spravované aplikace z webu Microsoft Azure Marketplace.

## <a name="create-from-the-marketplace"></a>Vytvoření z Marketplace

Nasazení spravované aplikace z Marketplace je podobná nasazení jakýmikoli prostředky z Marketplace. 

Na portálu, vyberte **+ nový** a vyhledejte typu řešení pro nasazení. Z dostupných možností vyberte, které potřebujete.

![hledání řešení](./media/managed-application-consume-marketplace/search-apps.png)

Zkontrolujte souhrn aplikace a vyberte **vytvořit**.

![Vytvoření spravované aplikace](./media/managed-application-consume-marketplace/create-marketplace-managed-app.png)

Jako žádného jiného řešení se zobrazí pole k zadání hodnot pro. Tato pole se liší podle typu spravované aplikace, které vytvoříte. 

## <a name="view-support-information"></a>Zobrazení informací o podporu

Po nasazení ve spravované aplikaci má zobrazte informace o podpoře pro aplikaci. V okně spravované aplikace je uvedena informace podpory.

![Podpora](./media/managed-application-consume-marketplace/support.png)

## <a name="view-publisher-permissions"></a>Zobrazení vydavatele oprávnění

Výrobce, který spravuje vaše aplikace je udělen přístup k prostředkům. Pokud chcete zobrazit tato oprávnění, vyberte **autorizací**.

![Povolení](./media/managed-application-consume-marketplace/authorizations.png)
