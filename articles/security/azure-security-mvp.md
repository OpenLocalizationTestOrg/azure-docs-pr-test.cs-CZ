# <a name="azure-security-mvp-program"></a>Program MVP zabezpečení Azure
Microsoft profesionály (MVP) jsou žebříčky komunity, kteří úspěšně demonstrovali své EXEMPLÁRNÍ snahy pomáhá ostatním využívat naplno své zkušenosti s technologiemi společnosti Microsoft. Jejich jejich nadšení, praktické znalosti a technických otázek a sdílet s komunitou se společností Microsoft.

Microsoft Azure teď rozpozná komunity odborníky s odbornými znalostmi speciální v zabezpečení Azure. MVPs Microsoft můžete být udělena MVP v Microsoft Azure v oblasti příspěvku zabezpečení Azure.

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

Když neexistuje žádné srovnávacího testu pro částečně stal MVP, protože se liší podle technologie a je životního cyklu, některá kritéria zahrnuje dopad kandidát příspěvky k online fóra, jako je například Microsoft Answers, TechNetu a MSDN; Wikiweby a online obsahu; konference a skupiny uživatelů; plynule přenášená data, webů, blogů a sociálních médií; a články a knih.

Jste odborník v zabezpečení Azure? Znáte někoho, kdo? Potom [jmenovat sobě nebo někomu jinému](https://mvp.microsoft.com/Nomination/nominate-an-mvp) k zabezpečení Azure MVP ještě dnes!
