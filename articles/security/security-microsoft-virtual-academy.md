# <a name="azure-security-courses-from-microsoft-virtual-academy"></a>Kurzy zabezpečení Azure z webu Microsoft Virtual Academy

Zahrnuty jsou postupy pro vývoj a nasazení lekce na zabezpečení.

Tyto ovládací prvky v rozsahu od šifrování dat na rest, správu klíčů, ochranu virtuálního počítače, protokolování a monitorování, ke službám proti malwaru, správu identit, přístupu ovládací prvky a další.
