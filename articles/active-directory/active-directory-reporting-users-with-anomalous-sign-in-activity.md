<ul><li>Přihlášení z neznámých zdrojů</li>
<li>Přihlášení po několika neúspěších</li>
<li>Přihlášení z více geografických poloh</li>
<li>Přihlášení z IP adres s podezřelou aktivitou</li>
<li>Přihlášení z možných nakažených zařízení</li>
<li>Nestandardní přihlašovací aktivita</li>
</ul>
