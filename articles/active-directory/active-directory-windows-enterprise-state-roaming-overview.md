* <span data-ttu-id="e587e-111">**Rozšířené zabezpečení** – Data se šifrují automaticky před opuštěním uživatele zařízení s Windows 10 pomocí Azure Rights Management (Azure RMS) a data zůstává zašifrovaný v klidovém stavu uložených v cloudu.</span><span class="sxs-lookup"><span data-stu-id="e587e-111">**Enhanced security** – Data is automatically encrypted before leaving the user’s Windows 10 device by using Azure Rights Management (Azure RMS), and data stays encrypted at rest in the cloud.</span></span> <span data-ttu-id="e587e-112">Veškerý obsah zůstává zašifrovaný v klidovém stavu uložených v cloudu, s výjimkou obory názvů, jako jsou názvy nastavení a aplikace systému Windows.</span><span class="sxs-lookup"><span data-stu-id="e587e-112">All content stays encrypted at rest in the cloud, except for the namespaces, like settings names and Windows app names.</span></span>  
* <span data-ttu-id="e587e-113">**Lepší správa a sledování** – poskytuje kontrolu a přehled, přes který se má synchronizovat nastavení ve vaší organizaci a v zařízení, která díky integraci portálu Azure AD.</span><span class="sxs-lookup"><span data-stu-id="e587e-113">**Better management and monitoring** – Provides control and visibility over who syncs settings in your organization and on which devices through the Azure AD portal integration.</span></span>
