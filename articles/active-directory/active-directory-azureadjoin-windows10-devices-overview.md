# <a name="windows-10-for-the-enterprise-ways-to-use-devices-for-work"></a>Windows 10 pro firmy: Možnosti, jak používat zařízení pro práci
Windows 10 vám dává možnost pomocí služby Azure Active Directory (Azure AD). Zařízení s Windows 10 můžete připojit ke službě Azure AD, aby uživatelé mohli podepsat systému Windows pomocí účty Azure AD nebo přidáním jejich ID Azure získat přístup k obchodním aplikacím a prostředkům.
