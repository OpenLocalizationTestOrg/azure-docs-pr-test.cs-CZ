# <a name="azure-ad-and-applications-assigning-users-to-an-application"></a>Azure AD a aplikace: přiřazování uživatelů k aplikaci
Než uživatelé a skupiny můžete přiřadit k aplikaci, musí vyžadovat přiřazení uživatelů.  Další informace o tom, jak vyžadovat přiřazení uživatelských najdete [nutnosti přiřazení uživatelských](active-directory-applications-guiding-developers-requiring-user-assignment.md) článku.

## <a name="assigning-users-to-an-application"></a>Přiřazování uživatelů k aplikaci
1. Přihlaste se k portálu Azure pomocí účtu správce.
2. Klikněte na **všechny položky** položku v hlavní nabídce.
3. Vyberte adresář, který používáte pro aplikaci.
4. Klikněte na **aplikace** kartě.
5. Vyberte aplikaci ze seznamu aplikací přidružených k tomuto adresáři.
6. Klikněte **uživatele a skupiny** kartě.
7. Vyberte uživatele, které chcete přiřadit k aplikaci.
8. Klikněte na tlačítko **přiřadit**.
9. Klikněte na tlačítko **Ano** po zobrazení výzvy.
