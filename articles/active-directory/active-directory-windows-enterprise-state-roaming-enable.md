## <a name="data-retention"></a><span data-ttu-id="fbe42-132">Uchovávání dat</span><span class="sxs-lookup"><span data-stu-id="fbe42-132">Data retention</span></span>

<span data-ttu-id="fbe42-139">**Odstranění dat zastaralých**: Data, která nepoužil, jeden rok ("dobu uchování"), budou považovány za zastaralé a budou odstraněny z Azure.</span><span class="sxs-lookup"><span data-stu-id="fbe42-139">**Stale data deletion**: Data that has not been accessed for one year (“the retention period”) will be treated as stale and may be deleted from Azure.</span></span> <span data-ttu-id="fbe42-140">Doba uchovávání se může změnit, ale nesmí být menší než 90 dnů.</span><span class="sxs-lookup"><span data-stu-id="fbe42-140">The retention period is subject to change but will not be less than 90 days.</span></span> <span data-ttu-id="fbe42-141">Zastaralá data mohou být konkrétní sadu nastavení Windows/aplikace nebo všechna nastavení pro uživatele.</span><span class="sxs-lookup"><span data-stu-id="fbe42-141">The stale data may be a specific set of Windows/application settings or all settings for a user.</span></span> <span data-ttu-id="fbe42-142">Například:</span><span class="sxs-lookup"><span data-stu-id="fbe42-142">For example:</span></span>
