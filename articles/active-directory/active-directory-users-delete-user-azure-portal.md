# <a name="delete-a-user-from-a-directory-in-azure-active-directory"></a>Odstranit uživatele z jiného adresáře v Azure Active Directory
Tento článek vysvětluje, jak toodelete uživatele z jiného adresáře v Azure Active Directory (Azure AD). Informace o přidání nového uživatele tooyour organizace najdete v tématu [přidat nové uživatele tooAzure služby Active Directory](active-directory-users-create-azure-portal.md).

## <a name="toodelete-a-user"></a>toodelete uživatele
1. Přihlaste se příliš[hello portál Azure](https://portal.azure.com) pomocí účtu, který je globálním správcem adresáře hello.
2. Vyberte **další služby**, zadejte **uživatelů a skupin** v hello textového pole a pak vyberte **Enter**.

   ![Správa uživatelů otevírání](./media/active-directory-users-delete-user-azure-portal/create-users-user-management.png)
3. Na hello **uživatelů a skupin** vyberte **uživatelé**.

   ![Okno Uživatelé otevírání hello](./media/active-directory-users-delete-user-azure-portal/create-users-open-users-blade.png)
4. Na hello **uživatelé a skupiny - Uživatelé** okně vyberte uživatele ze seznamu hello.
5. V okně hello hello vybraného uživatele, vyberte **přehled**a pak v řádku nabídek hello vyberte **odstranit**.

    ![Výběr příkaz Delete hello](./media/active-directory-users-delete-user-azure-portal/create-users-delete-command.png)
