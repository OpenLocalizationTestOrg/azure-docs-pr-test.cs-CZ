# <a name="manage-programs-and-their-controls"></a>Správa programů a jejich ovládací prvky 

Azure Active Directory (Azure AD) zahrnuje přístup recenze členy skupiny a přístup k aplikaci. Tyto příklady ovládacích prvků zajistěte dohledu pro kdo má přístup k aplikacím a členství ve skupinách vaší organizace. Organizace může pomocí těchto ovládacích prvků efektivně adres jejich zásad správného řízení, řízení rizik a požadavky na dodržování předpisů.

## <a name="create-and-manage-programs-and-their-controls"></a>Vytvářet a spravovat programů a jejich ovládací prvky
Můžete zjednodušit postupy ke sledování a shromažďování recenze přístupu pro různé účely jejich uspořádáním do programy. Každý kontrola přístupu může být propojený program. Potom při přípravě sestavy pro auditor, pouze recenze přístupu v oboru pro konkrétní initiative jsou viditelné.

Chcete-li zobrazit seznam programů, přejděte na [přístupu zkontroluje stránky](https://portal.azure.com/#blade/Microsoft_AAD_ERM/DashboardBlade/) a vyberte **programy**.

**Výchozí Program** je vždy k dispozici. Pokud jste v roli globálního správce, můžete vytvořit další programy. Například můžete mít jeden program pro každou initiative dodržování předpisů nebo obchodní cíle.

Pokud již nepotřebujete program a všechny ovládací prvky na sebe napojenou nemá, můžete ji odstranit.
