# <a name="how-to-stop-being-billed-for-azure-remoteapp"></a>Postup zastavení se účtují pro Azure RemoteApp
> [!IMPORTANT]
> Azure RemoteApp se přestává používat dne 31. srpna 2017. Podrobnosti najdete v tomto [oznámení](https://go.microsoft.com/fwlink/?linkid=821148).
> 
> 

Tak dlouho, dokud máte kolekce Azure Remoteappu, i když bez přidělených uživatelů, budou platit pro vzdálenou aplikaci RemoteApp. Pokud chcete zastavit tento fakturace, odstraňte všechny kolekce. 

Chcete-li odstranit kolekce:

1. Na portálu Azure classic otevřete stránku vzdálené aplikace RemoteApp.
2. Kliknutím na kolekci, kterou chcete odstranit a potom klikněte na **odstranit** v dolní části stránky.
3. Tento postup opakujte pro každou kolekci v rámci vašeho předplatného. 

Může trvat několik minut odstranit každou kolekci.
