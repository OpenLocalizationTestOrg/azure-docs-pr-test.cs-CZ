# <a name="whats-coming-in-azure-remoteapp"></a>Co Připravujeme v Azure Remoteappu
> [!IMPORTANT]
> Azure RemoteApp se přestává používat dne 31. srpna 2017. Podrobnosti najdete v tomto [oznámení](https://go.microsoft.com/fwlink/?linkid=821148).
> 
> 

**Zrušili všechny položky preview a plán a ukončeno z důvodu oznámené vyřazení služby Azure RemoteApp.**

| Funkce | Status |
| --- | --- | --- |
| Vzdálené aplikace Azure k dispozici v obou [portál Azure classic](http://manage.windowsazure.com) a [portál Azure](https://portal.azure.com) -privátní Preview verzi |**Došlo ke zrušení** |
| Možnost publikování jednotlivých aplikací konkrétním uživatelům – privátní Preview verzi ([zpětné vazby](https://feedback.azure.com/forums/247748-azure-remoteapp/suggestions/6067043-allow-the-ability-to-publish-specific-apps-to-spec/)) |**Došlo ke zrušení** |
| Balíček Instalační služby MSI redistributable klienta systému Windows ([zpětné vazby](https://feedback.azure.com/forums/247748-azure-remoteapp/suggestions/6627191-client-deployment-provide-an-msi-package-to-allo/))-privátní Preview verzi |**Došlo ke zrušení** |
| Podpora Azure Resource Manager (ARM) |**Došlo ke zrušení** |
| Podpora prostředků Azure v2 |**Došlo ke zrušení** |
| Cloud Solution Provider (CSP) |**Došlo ke zrušení** |
| Dostupnost v oblasti Indie |**Došlo ke zrušení** |
| Dostupnost v oblasti Spojené království |**Došlo ke zrušení** |

Další vylepšení přidali na začátku roku:

* Klienta na základě HTML5 pro Azure RemoteApp
* Univerzální aplikace Windows 10 s poměrně pro Azure RemoteApp – náhled
* Nasazení do virtuální sítě bez připojení k doméně (cloudové kolekce připojení k virtuálním počítačům Azure)
* Znovu použít existující virtuální síť pro hybridní kolekci vzdálené aplikace RemoteApp (také umožňuje ExpressRoute)
* ExpressRoute
* Rutin RemoteApp prostředí PowerShell a sady SDK pro správu a nasazení
* Vytvořit a odeslat bitovou kopii virtuálního počítače Azure
* Nahrát-obchodní (LOB) aplikace do kolekce cloudu, a také v hybridní kolekce
* Podpora pro Windows Phone
* Řídicí panel metriky využití
* Zřídit velký počet uživatelů pomocí souboru CSV
* Dostupnost v australské datových centrech
