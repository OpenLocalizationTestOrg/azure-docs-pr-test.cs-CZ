# <a name="best-practices-for-azure-remoteapp-clients"></a>Osvědčené postupy pro klienty Azure RemoteApp
> [!IMPORTANT]
> Azure RemoteApp se přestává používat dne 31. srpna 2017. Podrobnosti najdete v tomto [oznámení](https://go.microsoft.com/fwlink/?linkid=821148).
> 
> 

Tyto informace můžete použít Azure RemoteApp klienty:

* Vždy používejte nejnovější verzi klienta. Tím se zajistí, že má verze klienta, které používáte nejnovější opravy chyb a vylepšení funkcí. Budete se muset přihlásit k odběru automaticky aktualizace pro klienta do příslušné úložiště.
* Vzdálené aplikace RemoteApp se automaticky odhlásit můžete Pokud jste po určitou dobu neaktivní. Aby nedošlo ke ztrátě dat, doporučujeme zavření aplikace po ukončení používání služby.
