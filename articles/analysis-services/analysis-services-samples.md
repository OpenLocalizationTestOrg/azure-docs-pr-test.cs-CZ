# <a name="azure-analysis-services-samples"></a>Ukázek Azure Analysis Services
Pomocí následující ukázkové prostředky můžete další informace o a testování služby Analysis Services pro vaše prostředí.

## <a name="code-samples"></a>Ukázky kódů
[Služby Analysis Services](https://github.com/Microsoft/Analysis-Services) úložišti na Githubu zahrnuje projekty komunity a ukázky kódu s otevřeným zdrojem. 

## <a name="tabular-model-project-and-database-samples"></a>Ukázky projektu a databázi tabulkového modelu
[Adventure Works pro službu Analysis Services](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks-analysis-services) na Githubu je nejčastěji používanou ukázkový projekt tabulkový model. Můžete si stáhnout projektu VS nebo úplnou vzorovou databázi tabulkového modelu.


## <a name="sample-databases-on-github"></a>Ukázkové databáze na Githubu
Tyto ukázkové databáze na Githubu můžete použít pro vytváření a testování vlastních modelů. 

[Adventure Works ukázkové databáze](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks2014)

[Wide World Importers k účetnictví ukázkové databáze](https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0)

## <a name="adventure-works-internet-sales-sample-model-in-azure-portal"></a>Prodej Internet společnosti Adventure Works ukázkový model na portálu Azure
Pokud máte serveru služby Analysis Services na portálu Azure, můžete rychle a snadno vytvářet ukázkový model. Na stránce Přehled váš server na portálu Azure, klikněte na tlačítko **nový model**a potom v **zvolit zdroj dat**, vyberte **vzorová data**.
