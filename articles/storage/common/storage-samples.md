Pomocí níže uvedených odkazů pro zobrazení a stáhněte si ukázky kódu Azure Storage a aplikace.

## <a name="net-samples"></a>Ukázky .NET

* [Ukázek Azure Storage pomocí rozhraní .NET](storage-samples-dotnet.md)

## <a name="java-samples"></a>Ukázky jazyka Java

* [Ukázek Azure Storage používá Java](storage-samples-java.md)

## <a name="nodejs-samples"></a>Ukázky Node.js

* [Proces pro načtení objektů BLOB](https://github.com/Azure/azure-storage-node/tree/master/examples/blobuploader)
* [Nahrávání a stahování objektů blob](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/blobuploaddownloadsample.js)
* [Token pro pokračování](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/continuationsample.js)
* [Zásady opakování](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/retrypolicysample.js)
* [Sdílený přístupový podpis](https://github.com/Azure/azure-storage-node/blob/master/examples/samples/sassample.js)
