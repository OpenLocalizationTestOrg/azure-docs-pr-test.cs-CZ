# <a name="using-the-azure-importexport-tool-classic-deployment-model"></a>Pomocí nástroje Azure Import/Export (modelu nasazení classic)

Nástroj Azure Import/Export (WAImportExport.exe) slouží k vytvoření a Správa úloh pro službu Azure Import/Export umožňuje přenos velkých objemů dat do nebo z Azure Blob Storage.

Tato dokumentace je pro model nasazení classic nástroj Azure Import/Export. Informace o používání nejnovější verzi nástroje najdete v tématu [pomocí nástroje Azure Import/Export](../storage-import-export-tool-how-to.md).

V následujících článcích ukazují, jak na:

- Instalace a nastavení nástroj importu a exportu.
- Připravte pevné disky pro úlohu, kde můžete importovat data z jednotky do úložiště objektů Blob Azure.
- Zkontrolujte stav úlohy s kopírovat soubory protokolu. 
- Opravte úlohu importu. 
- Opravte úlohu exportu. 
- Řešení potíží s nástroj Azure Import/Export, v případě, že došlo k potížím při během procesu.
