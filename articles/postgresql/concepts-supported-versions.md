# <a name="supported-postgresql-database-versions"></a>Podporované databázi PostgreSQL verze
Cílem společnosti Microsoft je podporují n-2 verze modulu PostgreSQL v databázi Azure služby PostgreSQL, což znamená aktuálně vydaná hlavní verzi (ne) a dvě předchozí verze hlavní-(2).

Azure databázi PostgreSQL aktuálně podporuje následující verze:

## <a name="postgresql-version-962"></a>Verze PostgreSQL 9.6.2
Odkazovat [PostgreSQL dokumentace](https://www.postgresql.org/docs/9.6/static/release-9-6-2.html) Další informace o vylepšení a opravy v PostgreSQL 9.6.2.

## <a name="postgresql-version-957"></a>Verze PostgreSQL 9.5.7
Odkazovat [PostgreSQL dokumentace](https://www.postgresql.org/docs/9.5/static/release-9-5-7.html) Další informace o vylepšení a opravy v PostgreSQL 9.5.7.

## <a name="managing-updates-and-upgrades"></a>Správa aktualizací a upgradů
Azure databázi PostgreSQL automaticky spravuje opravy pro vedlejší verze aktualizace. V současné době ve verzi public preview, nepodporuje upgrade hlavní verze. Například upgrade z PostgreSQL 9.5 PostgreSQL 9.6 není podporován.
