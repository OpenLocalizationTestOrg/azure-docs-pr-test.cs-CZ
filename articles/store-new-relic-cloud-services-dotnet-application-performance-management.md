# <a name="new-relic-application-performance-management-on-azure"></a>Nové Správa výkonu aplikací New Relic v Azure
New Relic špičkových výkonu můžete přidat monitorování do vaší Azure hostované aplikace. Spolu s komplexní funkcemi pro monitorování, řešení potíží a ladění aplikace Azure máte také nárok na sníženou cenu produktů New Relic pomocí služby Azure.

## <a name="what-is-new-relic"></a>Co je to New Relic?
S [New Relic produkty](https://newrelic.com/products), můžete vyřešit chyby aplikace, předcházení potenciální problémy a pochopit výkon celé prostředí. Je navržený tak, aby ušetřil čas při identifikaci a diagnostikování potíží s výkonem a abyste měli informace, které k jejich řešení potřebujete, jednoduše na dosah ruky.

New Relic sleduje doba načítání a propustnost pro webové transakce, jak ze serveru a do prohlížečů uživatelů. Zobrazuje, kolik času stráveného v databázi, analyzuje pomalé dotazy a webových požadavků, poskytuje provozu monitorování a výstrahy, sleduje výjimky aplikací a mnoho další. 

## <a name="special-pricing"></a>Speciální ceny
Nové standardní Relic je zdarma uživatelům Azure. Pro nové New Relic se nabízí podle velikost instance pro cloudové služby Azure. Informace o cenách najdete v článku [New Relic stránky](https://azure.microsoft.com/marketplace/partners/newrelic/newrelic/) v Azure Marketplace nebo kontaktujte New Relic (sales@newrelic.com) pro cenové úrovni celého podniku.

Azure zákazníci obdrží dvě týden zkušební předplatné Pro nový Relic, když nasadí agent New Relic.

## <a name="sign-up-for-new-relic-using-the-azure-store"></a>Zaregistrujte si New Relic pomocí úložiště Azure
New Relic se bezproblémově integruje se webové role Azure a pracovní role. Můžete rychle a snadno zaregistrovat New Relic přímo z úložiště Azure. Pokyny najdete v tématu [Azure ukládání pokyny pro přihlášení](https://docs.newrelic.com/docs/agents/net-agent/azure-installation/azure-cloud-services#signup) z New Relic.

## <a name="view-your-data"></a>Zobrazení dat
Jakmile jste zaregistrovali, můžete využít výhod úžasné monitorování aplikací a datové analýzy New Relic. Chcete-li zkontrolovat výkon vaší aplikace v New Relic:

1. Z portálu Azure vyberte spravovat.
2. Přihlaste se pomocí vaší e-mailu účet New Relic a heslo.
3. Vyberte svou aplikaci z aplikace index pro zobrazení dat všechny aplikace v [stránku přehled APM](https://docs.newrelic.com/docs/apm/applications-menu/monitoring/apm-overview-page).
