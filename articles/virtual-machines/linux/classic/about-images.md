## <a name="working-with-images"></a>Práce s obrázky
Azure rozhraní příkazového řádku (CLI) pro Mac, Linux a Windows můžete použít ke správě bitových kopií, která je k dispozici k předplatnému Azure. Také můžete použít portál Azure pro některé úlohy bitové kopie, ale příkazového řádku získáte další možnosti.

Příklady použití nástroje najdete v tématu [společné rozhraní příkazového řádku Azure na Linuxu a Macu](../cli-manage.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).
