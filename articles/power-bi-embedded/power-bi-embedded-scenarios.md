# <a name="common-power-bi-embedded-scenarios"></a>Běžné scénáře Power BI Embedded
**Microsoft Power BI Embedded** se zaměřuje na vystavení většina stávajících funkcí rozhraní API Power BI jako součást služby Azure pro vývoj aplikací s.  Kromě toho bude moci prostřednictvím kódu programu zřizovat, vytvořte a nasaďte potřebné prostředky a obsah Power BI. Další informace najdete v tématu [Začínáme s Microsoft Power BI Embedded](power-bi-embedded-get-started.md).

Tady je několik scénářů pro použití **Power BI Embedded** ve svých aplikacích.

* V rámci vlastní aplikace pomocí nového modelu ověření pomocí tokenu aplikace lze jej vkládat plně interaktivní sestavy, které byly vytvořeny v Power BI Desktop.
* Nezávislí dodavatelé softwaru a zákazníky, kteří vyvíjejí zákazníků, kterým čelí aplikace můžete použít **Power BI Embedded** služby a Power BI SDK, vložit interaktivní sestavy. Další informace, jak zjistit, [Začínáme s Microsoft Power BI Embedded](power-bi-embedded-get-started.md).
* Jako vývojář slouží k vytvoření vlastních vizualizací, které lze použít ve vaší vlastní aplikaci vizualizace rozhraní Power BI. V tématu [rozšířit Power BI s vlastní vizuály](https://powerbi.microsoft.com/custom-visuals/).
