# <a name="change-your-azure-account-profile-information-such-as-contact-email-address-and-phone-number"></a>Změnit informace z vašeho profilu účtu Azure například kontaktní e-mailová adresa a telefonní číslo

E-mailovou adresu v profilu správce účtu se společností Microsoft používá k upozornění na důležité aktualizace týkající se fakturace o předplatném. Doporučujeme vám, že zadáváte kontaktní e-mailovou adresu, která pravidelně kontrolovat.

## <a name="how-to-change-your-profile-information"></a>Postup změny informace z vašeho profilu

1. Přihlaste se k [centra účtů Azure](https://account.azure.com/Profile).
1. Vyberte **upravit podrobnosti** tlačítko a pak aktualizujte **profil** informace.

   ![Profil](./media/billing-how-to-change-azure-account-profile/profile.png)

## <a name="frequently-asked-questions"></a>Nejčastější dotazy

### <a name="can-notifications-be-sent-to-a-different-email-address-other-than-the-account-administrator-email-address-associated-with-my-account"></a>Můžete odesílat oznámení na jinou e-mailovou adresu jiných než e-mailovou adresu účtu správce, přidružené Můj účet?

Ano. E-mailovou adresu na účet získá důležitá oznámení o všechna předplatná v rámci účtu. Proto doporučujeme, aby je zadat kontaktní e-mailovou adresu, která pravidelně kontroluje správce účtu.

### <a name="can-i-change-the-account-administrator-email-address-in-my-profile"></a>Lze změnit v Můj profil e-mailová adresa správce účtu?

Ano. Správce účtu je osoba, která nastavila účet Azure a kdo obdrží důležité e-mailové oznámení o všechna předplatná v rámci účtu. Doporučujeme vám, že zadáváte kontaktní e-mailovou adresu, která pravidelně kontroluje správce účtu.

### <a name="does-updating-my-profile-email-also-update-my-login-email-address"></a>Aktualizace můj profil e-mailu také aktualizovat Moje přihlášení e-mailovou adresu?

Ne. Aktualizace profil e-mailovou adresu neaktualizuje e-mailovou adresu pro přihlášení. Chcete-li změnit e-mailovou adresu pro přihlášení, budete muset převést vlastnictví účtu.

### <a name="does-updating-my-profile-address-also-update-my-credit-card-billing-address"></a>Adresa Můj profil se aktualizuje také aktualizovat Moje platební karty fakturační adresu?

Další postup aktualizace fakturační informace najdete v tématu [změnit z platební karty použít k platbě za předplatné Azure](billing-how-to-change-credit-card.md).

### <a name="why-cant-i-update-the-country"></a>Proč nelze aktualizovat zemi?

Z důvodu technická omezení nemůžeme změnit země pro existující účet. Můžete však vytvořit účet v požadované země a pak požádejte podporu Azure o migraci vašich služeb k tomuto účtu.

### <a name="what-email-address-does-the-azure-billing-alerts-preview-feature-use"></a>Jakou e-mailová adresa nemá výstrahy fakturace Azure preview použijte funkci?

Obdržíte oznámení o výstrahách v e-mailovou adresu, že přidáte při registraci pro výstrahy. Je oddělená od e-mailová adresa správce účtu je e-mailová adresa použitá pro výstrahy. Další informace najdete v tématu [nastavit výstrahy pro vaše předplatné Microsoft Azure billing](billing-set-up-alerts.md).
