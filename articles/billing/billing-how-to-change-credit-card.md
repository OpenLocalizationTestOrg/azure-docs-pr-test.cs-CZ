# <a name="change-the-credit-card-used-to-pay-for-an-azure-subscription"></a>Změna z platební karty použít k platbě za předplatné Azure
Můžete přidat, změnit nebo odebrat platební karty, který je spojen s vaším účtem Azure při přihlášení k [centra účtů Azure](https://account.windowsazure.com/Subscriptions) jako [správce účtu](billing-subscription-transfer.md#whoisaa). 
 
<a id="addcard"></a>
## <a name="add-a-credit-card-as-a-payment-method"></a>Přidat platební kartu jako způsob platby

1. Přihlaste se k [centra účtů Azure](https://account.windowsazure.com/Subscriptions) jako správce účtu.
2. Vyberte předplatné.
3. Na pravé straně stránky vyberte **Spravovat způsoby platby**.
4. Vyberte "+" Přidat platební karty.
5. Vyberte typ platební karty a zadejte požadované informace.
6. Vyberte **Další**. 

Pokud karta je platný, se přidá do seznamu způsoby platby k dispozici.

Pokud dojde k chybě po přidání platební karty, přečtěte si téma [platební karty odmítl v Azure registrace](billing-credit-card-fails-during-azure-sign-up.md).

## <a name="edit-payment-information-for-an-existing-credit-card"></a>Úprava platebních údajů existující platební karty
  Pokud získá obnovit platební karty a číslo zůstane stejný, aktualizujte existující platební karty podrobnosti, třeba datum vypršení platnosti. Pokud změny číslo platební karty protože karta dojde ke ztrátě nebo odcizení nebo jejichž platnost vypršela, postupujte podle kroků v [přidat platební kartu jako způsob platby](#addcard) části. 

1. Přihlaste se k [centra účtů Azure](https://account.windowsazure.com/Subscriptions) jako správce účtu.
2. Vyberte odběr, který je propojený na platební kartu.</br> ![Snímek obrazovky, který znázorňuje předplatné příklad vybrali.](./media/billing-how-to-change-credit-card/selectsub.png)
3. Vyberte **spravovat způsoby platby**.</br> ![Snímek obrazovky zobrazující spravovat platební metody možnost.](./media/billing-how-to-change-credit-card/changesub_new.png)
4. Vyberte způsob platby.
5. Vyberte **upravit** vedle číslo platební karty.</br> ![Snímek obrazovky, který ukazuje možnost Upravit vedle platby.](./media/billing-how-to-change-credit-card/editcard_new.png)
6. Zkontrolujte, zda že je správný typ a karty číslo platební karty.
7. Vyberte **Další**.

## <a name="change-the-credit-card-that-pays-your-azure-bill"></a>Změna z platební karty, kterou platí faktury Azure

1. Přihlaste se k [centra účtů Azure](https://account.windowsazure.com/Subscriptions) jako správce účtu.
2. Vyberte předplatné.
3. Na pravé straně stránky vyberte **Spravovat způsoby platby**.
4. Klikněte na tlačítko **místo toho použít** a vyberte z platební karty, které chcete použít.

## <a name="remove-a-credit-card-from-the-account"></a>Odebrat platební karty z účtu
1. Přihlaste se k [centra účtů Azure](https://account.windowsazure.com/Subscriptions) jako správce účtu.
2. Vyberte předplatné.
3. Na pravé straně stránky vyberte **Spravovat způsoby platby**.
4. Klikněte na tlačítko **odstranit** pro platební karty, které chcete odstranit.

Pokud platební karty je spojen s další aktivní odběry Microsoft, nelze odebrat z účtu Azure. Odebrání z platební karty všechny aktivní odběry, které máte s Microsoftem.

##  <a name="how-to-make-payments"></a>Jak provádět plateb

* Pokud jste nastavili kreditní nebo debetní karty jako váš způsob platby, je automaticky provedena platbu.
* Pokud používáte [faktury](https://azure.microsoft.com/pricing/invoicing/), odeslání vaší platební do umístění uvedených v dolní části faktury.
