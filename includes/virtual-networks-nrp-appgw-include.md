## <a name="application-gateway"></a>Application Gateway
Application Gateway poskytuje vyrovnávání řešení v závislosti na Vyrovnávání zatížení vrstvy 7 Azure spravované HTTP zatížení. Vyrovnávání zatížení aplikací umožňuje použití hello pravidla směrování pro provoz v síti založené na protokolu HTTP. 
<BR>

| Vlastnost | Popis |
| --- | --- |
| **backendAddressPools** |Hello seznam IP adres hello back-end serverů. uvedené Hello IP adresy by měly buď patřit toohello podsíť virtuální sítě, nebo musí být veřejné nebo virtuálními IP Adresami nebo privátní IP |
| **backendHttpSettingsCollection** |Každý fond má nastavení, jako je port, protokol a spřažení na základě souborů cookie. Tato nastavení jsou vázané tooa fond a jsou použité tooall servery v rámci fondu hello |
| **frontendPorts** |Toto je veřejný port hello otevírá ve hello application gateway. Provoz volá Tenhle port a potom získá přesměruje tooone hello back-end serverů |
| **httpListeners** |Naslouchací proces má front-end port, protokol (Http nebo Https, ty jsou malá a velká písmena) a název certifikátu SSL hello (Pokud se konfiguruje přesměrování zpracování SSL) |
| **requestRoutingRules** |Hello pravidlo váže hello naslouchací proces a hello back-end serveru fondu a definuje, které back-end serveru fondu hello provoz směrovat. Aktuálně lze použít pouze jako kruhového dotazování |
