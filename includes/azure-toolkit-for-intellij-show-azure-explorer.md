## <a name="displaying-hello-azure-explorer-tool-window"></a>Zobrazení hello okno Průzkumníka nástroje Azure

Pokud hello **Azure Explorer** okno nástroje není zobrazená v IntelliJ, klikněte na tlačítko **zobrazení**, pak klikněte na tlačítko **nástroj Windows**a potom klikněte na **Azure Explorer**.
